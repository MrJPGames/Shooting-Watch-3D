TPSgui
======

TPS is a game where you press the A or A and B button as fast as possible!

Current Features:
-- Press A as fast in 10 seconds
-- See your HighScore
-- See your Average Score

Upcomming Features:
-- Save High/Average Score!
-- Have a watch feature
-- A possible A and B version
